1. Give two reasons on why it was easier for Watson to process clues in Jeopardy than it would be to process general language.
    • Clues have a much more standard structure
    • Clues are short
  2. Which of the following Natural Language Processing (NLP) steps did Watson skip?
    1. Recognizing speech
    2. Syntax analysis (grammar)
    3. Semantic analysis (meanings)
    4. Pragmatics (context)

    A. Since clues were given in written form and fed directly to Watson, Watson did not have to parse sounds to determine what words were being said.
  3. Which of the following is NOT a step for Natural Language Processing?
    1. Pragmatics
    2. Recognizing the language
    3. Semantic Analysis
    4. Punctuation Analysis
    5. Syntax Analysis

    D. According to Natural Language Processing the steps are as follows:

    1. Recognize the language (Watson does not do this, but it is considered a step)
    2. Syntax Analysis (Grammar)
    3. Semantic Analysis (Meanings)
    4. Pragmatics (Context)

    Punctuation would fall under the overall arch of syntax and is, on its own, not one of the steps for NLP.
