Exercises

Practice hard. Have fun!

This page contains extra practice exercises from previous offerings of this for you to work through to help ensure that you are understanding the material correctly. These exercises do not cover all topics discussed in class. They are meant to be a supplement to your studying. There are over 500 questions on this page so it is possible that a solutions file may contain an incorrect answer, so please, when in doubt, reach out to the teaching team. A mistake on the exercises page is not a justification for a wrong answer on the exam.

Exercises

One more note about how to use these exercises: the only way for them to do you good is for you to attempt the exercise without looking at the answer and then compare your work to the answer. It is easy to convince yourself that you understand the material by looking at the answers. This does not work. The only way to study is to actually do the problems. If you try a problem and can't do it without the solutions, find another, similar problem (or make one up) and make sure you can solve that.
