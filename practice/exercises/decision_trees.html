  3. True or false: A decision tree is a tree where nodes are conditions and edges are attributes that have to be met:

    False

    A decision tree has the nodes as attributes and edges as conditions. Whether you follow an edge depends on what the condition of the situation you are considering is. For example, if you had a node that had an attribute of "Shirt Colour" with outgoing edges labeled "Blue", "Red", and "Yellow", which edge you follow to the next node would depend on your current shirt colour (i.e., it is conditional).

  4. True or false: In a rooted tree data structure in computer science, nodes must be labelled for it to be a valid tree.

    False.

    The following conditions must be met in order for a data structure to be considered as a rooted tree:

    • There is only one root node (i.e., there is only one node in the tree that has no parent)
    • Each node can have zero or more children
    • All nodes (except for the root node) only have one parent
    • Edges between two nodes represent a parent/child relationship
    • Nodes and edges may be labelled (not a necessity)
  5. True or false: In a root tree data structure in computer science, there can only be one root.

  11. Which of the following is true about the decision trees?
    1. One of the common methods for deciding which attributes to split on is to try to reduce entropy
    2. In decision tree, the more you reduce entropy, the lower the information gain
    3. Decision tree is a tree where nodes are conditions and the edges are attributes that have to be met
    4. When we split the decision tree, we should make the resulting groups more heterogenous
    5. It doesn't make any difference whether we reduce entropy or enlarge entropy. the decision tree will look exactly the same

    A is right. chosing to reduce entropy will make the decision tree easier to work on and simpler.

    B is wrong because the more you reduce the entropy, the higher the information gain.

    C is wrong because in decision tree, the nodes are attributes and the edges are conditions.

    D is wrong because we should make the resulting groups more homogeneous when we split the decision tree. it will make the decision tree simpler and easier to observe, thus reducing the work that we have to do.

    E is wrong because reduce entropy is better than enlarge entropy. reduce entropy will make it easier to make and observe, while enlarge entropy will make it more complicated since we have to observe more nodes and edges in the decision tree.

  12. False windy weather had 3 yes's and 2 no's. True windy weather had 2 yes's and 5 no's. False sunny weather has 1 yes and 3 no's. True sunny weather has 4 yes's and 4 no's.

    Which type of weather has the most entropy?

    1. False windy weather
    2. True sunny weather
    3. False sunny weather
    4. True windy weather

    The entropy is the amount of disorder:

    • A → 3 +2 = 5 - less than 8
    • B → 4 + 4 = 8 - greatest entropy (because there are 8 yes's and no's)
    • C → 1 + 3 = 4 - less than 8
    • D → 2 + 5 = 7 - less than 8
