    For each number from 1 to 100:

    1. See if the number is divisible by both 3 and 5. If it is, print "FizzBuzz". Then, ignore the rest of the following steps and move on to the next number.
    2. See if the number is divisble by 3. If it is, print "Fizz". Then, ignore the rest of the following steps and move on to the next number.
    3. See if the number is divisble by 5. If it is, print "Buzz". Then, ignore the rest of the following steps and move on to the next number.
    4. Print out the number.

    Note that steps 2 and 3 can be switched with each other (i.e., you can test to see if the number is divisble by 5 first rather than 3).

  3. Given a word, check to see if it is a palindrome. Note: a palindrome is a word which reads the same forwards and backwards (e.g., racecar).
    1. Ask for the word to check.
    2. Take a marker (let's call it marker A) and point it at the first character of the word.
    3. Take another marker (let's call it marker B) and point it at the last character of the world.
    4. While the two markers are not pointing at the same character OR marker B is not pointing at a character that occurs before the character marker A is pointing at, repeat the following:
      1. Check to see if the characters marker A and B are pointing at are the same.
      2. If the characters are the same, move marker A forward to the next character and marker B backwards to the character that occurs prior to the character it is pointing at.
      3. If the characters are not the same, stop checking right away (since it will never be a palindrome).

  12. You are responsible for designing an algorithm that plays a number guessing game with the user. The algorithm will choose a random number from 1 to 100 and the user will guess a number.     If the user has guessed a number that is smaller than the number the algorithm has chosen, the algorithm should tell the user "The chosen number is larger!".
